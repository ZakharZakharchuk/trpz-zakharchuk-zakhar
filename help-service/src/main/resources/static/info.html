We are thrilled to announce some exciting new updates to our text editor that enhance your editing experience. These improvements are designed to make your workflow smoother and more efficient.

Among the key enhancements:

  • Snippets Management: We've introduced a new Snippets menu, allowing you to create, manage, and insert code snippets seamlessly. Boost your productivity with reusable code blocks.
