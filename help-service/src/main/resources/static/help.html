Text Editor Functionality

Edit Menu:

  • Cut: Select an area you want to cut and use the "Cut" option in the Edit menu.
  • Copy: Use the "Copy" option in the Edit menu to copy selected text.
  • Paste: Use the "Paste" option in the Edit menu to paste your text.
  • Undo: Use the "Undo" option in the Edit menu to undo operations (Cut, Paste, Find and replace, comment code).

Snippets submenu (in Edit Menu):

  • Create Snippet: Use the "Create Snippet" option in the Snippets submenu to create a snippet.
  • Use Snippet: Use the "Use Snippet" option in the Snippets submenu to use a snippet.

Find and Replace (in Edit Menu):

Use the "Find and Replace" option in the Edit menu. Write the text you want to replace and then the text with which to replace. It will replace all matches in the text.

Comment Code (in Edit Menu):

Select the part of the code you want to comment and use the "Comment Code" option in the Edit menu.
